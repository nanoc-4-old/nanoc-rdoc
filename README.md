# nanoc-rdoc

This provides a [RDoc::Markup](http://rdoc.rubyforge.org/RDoc/Markup.html) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-rdoc`

## Usage

```ruby
filter :rdoc
```

This filter takes no options.
